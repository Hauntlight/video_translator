Guida all'Installazione e Uso di Video Translator

Questa guida ti accompagnerà nell'installazione e nell'utilizzo del tool. Inizia con il metodo di Installazione Rapida, che è il più semplice e consigliato per tutti gli utenti.

Fase 1: Installazione Rapida (Consigliata)

Questi script automatizzano l'installazione delle dipendenze di sistema e delle librerie Python. L'unico prerequisito è avere Python 3.11 già installato sul tuo sistema.

Windows

  1. Assicurati di aver installato Python 3.11 spuntando l'opzione "Add to PATH".
  2. Trova il file setup.bat nella cartella del progetto.
  3. Fai click con il tasto destro sul file e scegli "Esegui come amministratore".
  4. Lo script si occuperà di tutto. Al termine, segui le istruzioni a schermo per avviare il programma.

/ macOS e Linux

  1. Assicurati di aver installato Python 3.11.
  2. Apri il Terminale e naviga fino alla cartella del progetto.
  3. Rendi lo script eseguibile con il seguente comando: 
    chmod +x setup.sh
  4. Esegui lo script: 
    ./setup.sh
  5. Ti potrebbe essere richiesta la password di amministratore per installare i pacchetti. Al termine, segui le istruzioni a schermo.

Se l'installazione rapida fallisce, non preoccuparti. Puoi seguire i passaggi dettagliati nella sezione Installazione Manuale qui sotto.

Fase 3: Esecuzione e Utilizzo

Una volta completata l'installazione, sei pronto ad usare il tool.

  1. Avvia l'Applicazione: Assicurati che l'ambiente virtuale sia attivo e lancia il programma: 
    python main.py
  2. Seleziona il Video:
    • Clicca sul pulsante "Seleziona Video (.mp4)".
    • Scegli un file video dal tuo computer. L'applicazione controllerà automaticamente se hai già iniziato a lavorare su questo file.
  3. Flusso di Lavoro:
    1. Trascrizione: Clicca su "1. Trascrivi". Il tool estrarrà l'audio e lo convertirà in testo italiano. Questo è il passaggio più lungo. Al termine, il testo apparirà nell'editor.
    2. Revisione (Italiano): Correggi eventuali errori di trascrizione direttamente nell'area di testo. Clicca su "Salva Modifiche al Testo Corrente" per salvare il tuo lavoro.
    3. Traduzione: Clicca su "2. Traduci". Il testo italiano verrà tradotto in inglese e sostituirà il contenuto dell'editor.
    4. Revisione (Inglese): Allo stesso modo, puoi correggere la traduzione automatica per migliorarne la qualità. Ricorda di salvare le modifiche.
    5. Generazione Video: Clicca su "3. Genera Video". Il tool genererà l'audio inglese, lo sincronizzerà con i tempi originali e lo unirà al video originale (senza audio).
  4. Risultato Finale:
    • Troverai il video finale con audio in inglese nella cartella output.
    • Nella stessa cartella, troverai anche i file di testo con la trascrizione (_it.txt) e la traduzione (_en.txt) per poterli riutilizzare o modificare in futuro.
Funzione di Ripresa Automatica: Se selezioni un video per cui hai già generato una trascrizione o una traduzione, l'applicazione caricherà automaticamente lo stato più avanzato del tuo lavoro, permettendoti di ripartire da dove eri rimasto.
