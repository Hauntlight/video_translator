Guida all'Installazione e Uso di Video Translator

Questa guida ti accompagnerà passo dopo passo nell'installazione e nell'utilizzo del tool Video Translator. Segui attentamente le istruzioni specifiche per il tuo sistema operativo.

Fase 1: Installazione dei Prerequisiti

Prima di installare il tool, è necessario assicurarsi che sul tuo sistema siano presenti alcuni software fondamentali.

1.1. Python 3.11

Il progetto è sviluppato e testato con Python 3.11. Versioni diverse potrebbero non funzionare correttamente.

Windows

  1. Vai alla pagina di download ufficiale di Python 3.11.9.
  2. Scorri fino in fondo e scarica il file "Windows installer (64-bit)".
  3. Esegui l'installer. Nella prima schermata, assicurati di spuntare la casella "Add Python 3.11 to PATH". Questa opzione è fondamentale.
  4. Clicca su "Install Now" e segui le istruzioni.
  5. Verifica: Apri un nuovo Prompt dei comandi (CMD) e digita python --version. Dovresti vedere "Python 3.11.x".

macOS

  1. Il modo migliore per installare software su macOS è tramite Homebrew. Se non lo hai, apri il Terminale e installalo con questo comando: 
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Una volta installato Homebrew, installa Python 3.11: 
    brew install python@3.11
  3. Verifica: Apri un nuovo Terminale e digita python3 --version. Dovresti vedere "Python 3.11.x".

Linux (Debian/Ubuntu)

  1. Apri il Terminale e aggiorna la lista dei pacchetti: 
    sudo apt update && sudo apt upgrade
  2. Installa Python 3.11 e i pacchetti necessari per creare ambienti virtuali: 
    sudo apt install python3.11 python3.11-venv
  3. Verifica: Digita python3.11 --version. Dovresti vedere "Python 3.11.x".

1.2. FFmpeg

FFmpeg è uno strumento essenziale per l'elaborazione di file audio e video, usato da moviepy e pydub.

Windows

  1. Vai al sito gyan.dev e scarica la build "release full" (es. ffmpeg-release-full.7z).
  2. Estrai il contenuto dell'archivio in una cartella permanente, ad esempio C:\ffmpeg.
  3. Copia il percorso della cartella bin al suo interno (es. C:\ffmpeg\ffmpeg-7.0.1-full_build\bin).
  4. Aggiungi questo percorso al PATH di sistema:
    • Cerca "Modifica le variabili d'ambiente per l'account" nel menu Start.
    • Seleziona la variabile `Path` e clicca su "Modifica...".
    • Clicca su "Nuovo", incolla il percorso copiato e premi OK su tutte le finestre.
  5. Verifica: Apri un nuovo Prompt dei comandi e digita ffmpeg -version. Dovresti vedere le informazioni sulla versione di FFmpeg.

macOS

  1. Apri il Terminale e usa Homebrew per installare FFmpeg: 
    brew install ffmpeg
  2. Verifica: Digita ffmpeg -version.

Linux (Debian/Ubuntu)

  1. Apri il Terminale e installa FFmpeg: 
    sudo apt install ffmpeg
  2. Verifica: Digita ffmpeg -version.

1.3. Rubber Band Library

Questa è la libreria che alimenta pyrubberband per modificare la velocità dell'audio con alta qualità.

Windows

  1. Il modo più semplice è usare il gestore di pacchetti `winget`. Apri il Prompt dei comandi e digita: 
    winget install breakfast.rubberband
  2. Verifica: Apri un nuovo Prompt dei comandi e digita rubberband --version. Dovresti vedere le informazioni sulla versione.

macOS

  1. Apri il Terminale e usa Homebrew: 
    brew install rubberband
  2. Verifica: Digita rubberband --version.

Linux (Debian/Ubuntu)

  1. Apri il Terminale e installa il pacchetto della command-line interface: 
    sudo apt install rubberband-cli
  2. Verifica: Digita rubberband --version.

Fase 2: Installazione del Progetto

Con tutti i prerequisiti installati, possiamo ora configurare il progetto.

  1. Scarica il Progetto:
    • Opzione A (ZIP): Scarica il progetto come file ZIP e estrailo in una cartella a tua scelta.
    • Opzione B (Git): Se hai Git installato, clona il repository dal tuo terminale: 
      git clone <URL_DEL_REPOSITORY> VIDEO_TRANSLATOR
  2. Apri il Terminale nella Cartella del Progetto: 
    cd percorso/alla/cartella/VIDEO_TRANSLATOR
  3. Crea un Ambiente Virtuale: Questo isola le dipendenze del progetto. 
    python3 -m venv venv

    Su Windows, il comando potrebbe essere python -m venv venv.

  4. Attiva l'Ambiente Virtuale:
    • Windows: 
      venv\Scripts\activate
    • macOS/Linux: 
      source venv/bin/activate

    Dovresti vedere (venv) all'inizio della riga del terminale.

  5. Installa le Librerie Python: Questo comando leggerà il file requirements.txt e installerà tutto il necessario. 
    pip install -r requirements.txt

    Questa operazione potrebbe richiedere alcuni minuti, specialmente per il download del modello Whisper.

Fase 3: Esecuzione e Utilizzo

Una volta completata l'installazione, sei pronto ad usare il tool.

  1. Avvia l'Applicazione: Assicurati che l'ambiente virtuale sia attivo e lancia il programma: 
    python main.py
  2. Seleziona il Video:
    • Clicca sul pulsante "Seleziona Video (.mp4)".
    • Scegli un file video dal tuo computer. L'applicazione controllerà automaticamente se hai già iniziato a lavorare su questo file.
  3. Flusso di Lavoro:
    1. Trascrizione: Clicca su "1. Trascrivi". Il tool estrarrà l'audio e lo convertirà in testo italiano. Questo è il passaggio più lungo. Al termine, il testo apparirà nell'editor.
    2. Revisione (Italiano): Correggi eventuali errori di trascrizione direttamente nell'area di testo. Clicca su "Salva Modifiche al Testo Corrente" per salvare il tuo lavoro.
    3. Traduzione: Clicca su "2. Traduci". Il testo italiano verrà tradotto in inglese e sostituirà il contenuto dell'editor.
    4. Revisione (Inglese): Allo stesso modo, puoi correggere la traduzione automatica per migliorarne la qualità. Ricorda di salvare le modifiche.
    5. Generazione Video: Clicca su "3. Genera Video". Il tool genererà l'audio inglese, lo sincronizzerà con i tempi originali e lo unirà al video originale (senza audio).
  4. Risultato Finale:
    • Troverai il video finale con audio in inglese nella cartella output.
    • Nella stessa cartella, troverai anche i file di testo con la trascrizione (_it.txt) e la traduzione (_en.txt) per poterli riutilizzare o modificare in futuro.
Funzione di Ripresa Automatica: Se selezioni un video per cui hai già generato una trascrizione o una traduzione, l'applicazione caricherà automaticamente lo stato più avanzato del tuo lavoro, permettendoti di ripartire da dove eri rimasto.

Fase 4: Configurazione e Personalizzazione

Puoi personalizzare il comportamento del tool modificando il file config/settings.yaml con un qualsiasi editor di testo. Salva il file dopo ogni modifica.

4.1. Modificare il Modello di Trascrizione (Whisper)

Puoi bilanciare velocità e accuratezza della trascrizione scegliendo un modello Whisper diverso.

  • Apri config/settings.yaml e trova la sezione stt.
  • Modifica il valore di whisper_model.
stt:
  whisper_model: 'medium'

Modelli disponibili, dal più veloce al più accurato:

  • tiny: Più veloce, meno accurato.
  • base: Veloce, accuratezza di base.
  • small: Buon compromesso tra velocità e accuratezza.
  • medium: Consigliato. Ottima accuratezza, ma richiede più tempo.
  • large: Massima accuratezza, molto più lento e richiede più risorse (RAM/VRAM).

La prima volta che usi un nuovo modello, il programma dovrà scaricarlo. Questa operazione potrebbe richiedere tempo e una buona connessione a Internet.

4.2. Regolare il Rilevamento delle Pause

Se noti che le frasi vengono divise troppo o troppo poco, puoi modificare questi parametri nella sezione silence_detection.

silence_detection:
  min_silence_len: 700
  silence_thresh: -40
  • min_silence_len: La durata minima, in millisecondi, di una pausa affinché venga considerata come una separazione tra frasi. Aumenta questo valore se vuoi unire frasi separate da pause brevi.
  • silence_thresh: La soglia di "silenzio" in dBFS (Decibels relative to full scale). Valori più vicini allo zero (es. -30) richiedono un silenzio quasi assoluto, mentre valori più bassi (es. -50) sono più tolleranti al rumore di fondo.

4.3. Cambiare le Lingue di Traduzione

Puoi cambiare la lingua di partenza e di destinazione nella sezione languages.

languages:
  source: 'it'
  target: 'en'
  • source: La lingua dell'audio originale (es. 'it' per italiano). Specificare la lingua corretta aiuta Whisper a essere più accurato.
  • target: La lingua in cui tradurre il testo e generare il nuovo audio (es. 'en' per inglese, 'es' per spagnolo).
  • Puoi trovare i codici per le lingue in questa lista.

Risoluzione dei Problemi Comuni

  • Errore `ffmpeg not found` o `RuntimeWarning: Couldn't find ffmpeg`: FFmpeg non è installato correttamente o non è stato aggiunto al PATH di sistema. Rivedi il passo 1.2 e assicurati di aprire un nuovo terminale dopo aver modificato il PATH.
  • L'applicazione non parte o si chiude subito: Assicurati di aver attivato l'ambiente virtuale (venv) prima di lanciare python main.py. Controlla il file logs/app.log per messaggi di errore dettagliati.
  • La traduzione non funziona: Verifica la tua connessione a Internet, poiché il servizio di traduzione ne ha bisogno per funzionare.
  • Qualità audio rallentato scadente: Assicurati che la libreria Rubber Band sia installata correttamente (passo 1.3).
